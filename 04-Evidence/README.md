# Evidence Collection

## Overview

Evidence was collected from Splunk Enterprise after analyzing Windows Event Logs and Sysmon events generated during the simulated incident. The collected evidence was used to identify process execution details, verify user activity, and support the investigation findings.

---

## Evidence Collected

- Process Creation Events (Sysmon Event ID 1)
- Command Execution
- Parent Process Information
- User Account
- Computer Name
- Command Line Arguments
- Event Timestamp

---

## Purpose

The collected evidence was used to:

- Validate the investigation
- Support the incident timeline
- Identify Indicators of Compromise (IOCs)
- Document investigation findings
