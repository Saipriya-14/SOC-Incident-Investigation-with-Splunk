# Incident Report

## Incident Summary

A Windows endpoint generated Sysmon process creation events after PowerShell was executed. The activity was investigated using Splunk Enterprise to determine whether it was suspicious or part of legitimate administrative operations.

---

## Findings

- Command execution identified.
- Parent-child process relationship analyzed.
- Command-line activity reviewed.
- Process creation events verified.
- Timeline reconstructed.

---

## Conclusion

The investigation successfully demonstrated a SOC analyst workflow using Splunk Enterprise and Sysmon. Relevant evidence was collected, analyzed, and documented to support the investigation findings.
